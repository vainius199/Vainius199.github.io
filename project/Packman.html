packman

This is 2D clone of one of the classic game called Packman. Main goal here was to create game as clone as possible to that game. In the end i think it turned out pretty well and process of making it was fun.

The controls in this game are simple, just four arrow button (left, right, up, down). The goal in this game is to "eat" all packman food while avoiding enemies which have different movement patterns. When all food is gone, you will be moved to the next level. This game is endless until you lose all the lifes, after that the score will be saved and you can start again.
