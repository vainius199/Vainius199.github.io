Survive In VR

This is 3D game for android phones using AR (Vuforia) and VR (Google Cardboard) technolagies. Main goal here was to use paper picture as camera target for game main controller. This is my biggest game project so far.

To use paper controller you need to make sure that your phone camera can see it clearly. When it detects paper target, gun object in game will be activated and you can use it afterwards.

To use paper controller you can spin foward/backwards or left/right and character will move to that direction. To shoot from your gun you have to hide the right side of paper picture when its facing phone camera (as shown above), that will activate shooting function.
